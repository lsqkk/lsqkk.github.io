# 管理员登录与会话校验说明

## 功能概述
通过 `__API_BASE__/api/admin-auth` 校验管理员密码哈希并返回短期会话 token。前端保存 token 后，通过 `__API_BASE__/api/admin-verify` 校验会话有效性。

## 必要环境变量
- `ADMIN_PASSWORD_HASH`：管理员密码的 SHA-256 哈希
- `ADMIN_SESSION_SECRET`：会话签名密钥

## 前端登录流程示例
```javascript
async function adminLogin(passwordInputId = 'adminPassword', statusElId = 'adminStatus') {
  const input = document.getElementById(passwordInputId);
  const statusEl = document.getElementById(statusElId);
  const password = (input?.value || '').trim();
  if (!password) return;

  const passwordHash = await sha256(password); // 你的 sha256 实现
  const resp = await fetch('__API_BASE__/api/admin-auth', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ passwordHash })
  });
  const data = await resp.json();

  if (resp.ok && data.success && data.token) {
    localStorage.setItem('xjtu360_admin_token', data.token);
    if (statusEl) statusEl.textContent = '管理员已登录';
  } else {
    if (statusEl) statusEl.textContent = '密码错误';
  }
}

async function verifyAdminSession() {
  const token = localStorage.getItem('xjtu360_admin_token') || '';
  if (!token) return false;
  const resp = await fetch('__API_BASE__/api/admin-verify', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ token })
  });
  const data = await resp.json();
  return !!(resp.ok && data.valid);
}
```

## 注意事项
- 该 API 有来源校验，域名变更需同步更新 `api/admin-auth.js` 与 `api/admin-verify.js` 的允许列表。
- 建议把 token 存在 `localStorage`，退出时清理。
