# 网站设置与管理面板说明

## 网站设置页

`/settings` 是面向网站访问者的偏好设置页，使用 `LuxuryBackground` 背景。

当前支持：
- 默认语言：与顶栏语言选择共用 `quark_language_preference`
- 深浅色：`system` / `light` / `dark`
- 站点字体：读取 `assets/js/user-preferences.js` 中维护的字体列表，并写入 `--site-font-family`
- 动效强度与指针轨迹开关

偏好由 `assets/js/user-preferences.js` 统一管理，保存到 `localStorage` 的 `quark_site_preferences`。若用户已登录，会同步到 Firebase RTDB：

```text
user_activity/{uid}/settings
```

顶栏语言选择也会调用同一套偏好保存逻辑，因此登录用户在其他设备登录后会自动读取云端语言偏好。

## Quark 管理面板

`quark web` 会启动本地管理面板 Web UI；`quark config` 仍保留为兼容入口，二者当前指向同一套 Flask 应用。

默认地址：

```bash
quark web
# http://127.0.0.1:5050
```

管理面板可编辑以下 JSON 源目录：
- `src/config/json`
- `assets/pages/a`
- `assets/pages/blog`
- `assets/pages/games`
- `assets/pages/tool`

面板支持树形、代码、预览三种 JSON 编辑模式；保存前会由后端重新序列化 JSON。

当前可在 Web UI 执行的维护命令：
- `npm run build`
- `npm run typecheck`
- `npm run check:syntax`
- `quark updateposts`
- `quark updatelog`
- `git status --short`

此管理面板面向仓库管理者，仅应在本地可信环境运行，不作为公开站点页面部署。
