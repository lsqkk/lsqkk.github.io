## 🛠️ 技术特色

**这不是一个基于常见博客模板 + 静态网站生成器的项目**

- 🎯 **手搓开发**：从架构到组件均为自主设计与实现
- 🤖 **AI辅助编码**：在开发过程中与AI结对编程，探索新的开发范式
- 🎨 **极致自由度**：摆脱模板约束，每个细节都按心中所想实现
- 📚 **深度学习**：通过亲手构建，深入理解前端技术的每个层面

虽然这条路更具挑战，但带来的控制力和学习价值是无与伦比的。
