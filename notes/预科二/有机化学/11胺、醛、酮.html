胺、醛与酮

1. 胺类化合物

1.1 分类与命名

1.1.1 基本分类

  • 伯胺(1°胺)：RNH2_22​，如甲胺(CH3_33​NH2_22​)
  • 仲胺(2°胺)：R2_22​NH，如二甲胺((CH3_33​)2_22​NH)
  • 叔胺(3°胺)：R3_33​N，如三甲胺((CH3_33​)3_33​N)
  • 季铵盐：R4_44​N+^++X−^-−，如四甲基氯化铵((CH3_33​)4_44​N+^++Cl−^-−)

1.1.2 特殊胺类

  • 芳香胺：如苯胺(C6_66​H5_55​NH2_22​)
  • 杂环胺：
    • 吡咯(NHNHNH)
    • 吡啶(NNN)
    • 哌啶(NHNHNH)

1.1.3 命名规则

  1. IUPAC命名法：
    • 选择含氨基的最长碳链为主链
    • 编号使氨基位置最小
    • 后缀"-amine"

  1. 普通命名法：

    • 胺基作为取代基
    • 例：(CH3_33​)2_22​NH → 二甲胺

  2. 复杂胺类命名：

    • 使用"N"表示取代基位置
    • 例：C6_66​H5_55​NHCH3_33​ → N-甲基苯胺

1.2 物理性质

1.2.1 沸点与氢键

  • 伯胺和仲胺能形成分子间氢键(N-H···N)
  • 氢键强度：O-H···O > N-H···N
  • 沸点比较：
    • CH3_33​OH (65°C) > CH3_33​NH2_22​ (-6.3°C)
    • 正丁胺(78°C) > 二乙胺(55°C) > 叔丁胺(46°C)

1.2.2 溶解性

  • 低级胺(C1_11​-C5_55​)易溶于水
  • 高级胺水溶性降低
  • 芳香胺水溶性一般较差

1.2.3 其他性质

  • 多数胺类有特殊气味
  • 芳香胺通常为无色或淡黄色液体/固体

1.3 化学性质

1.3.1 碱性

  1. 碱性比较：

    • 脂肪胺 > 氨 > 芳香胺
    • 例：pKb_bb​值
      • 甲胺：3.36
      • 氨：4.75
      • 苯胺：9.38
      • 对硝基苯胺：13.0

  2. 影响碱性的因素：

    • 电子效应：供电子基增强碱性
    • 空间位阻：大位阻降低碱性
    • 溶剂化效应：水溶液中，伯胺>仲胺>叔胺

1.3.2 胺的制备

  1. 硝基化合物还原：

    • 催化氢化：
      R-NO2+3H2→Pd/CR-NH2+2H2O \text{R-NO}_2 + 3\text{H}_2 \xrightarrow{\text{Pd/C}} \text{R-NH}_2 + 2\text{H}_2\text{O} R-NO2​+3H2​Pd/C​R-NH2​+2H2​O
    • 金属还原：
      Ar-NO2+3Sn+6HCl→Ar-NH2+3SnCl2+2H2O \text{Ar-NO}_2 + 3\text{Sn} + 6\text{HCl} \rightarrow \text{Ar-NH}_2 + 3\text{SnCl}_2 + 2\text{H}_2\text{O} Ar-NO2​+3Sn+6HCl→Ar-NH2​+3SnCl2​+2H2​O

  2. 卤代烃氨解：
    R-X+NH3→R-NH2+HX \text{R-X} + \text{NH}_3 \rightarrow \text{R-NH}_2 + \text{HX} R-X+NH3​→R-NH2​+HX
    (可能产生混合胺类产物)

  3. 还原胺化：
    R2C=O+R’NH2→NaBH3CNR2CH-NHR′ \text{R}_2\text{C=O} + \text{R'NH}_2 \xrightarrow{\text{NaBH}_3\text{CN}} \text{R}_2\text{CH-NHR}' R2​C=O+R’NH2​NaBH3​CN​R2​CH-NHR′

1.3.3 胺的反应

  1. 烷基化反应：
    RNH2+R’X→RNHR′+HX \text{RNH}_2 + \text{R'X} \rightarrow \text{RNHR}' + \text{HX} RNH2​+R’X→RNHR′+HX
    (可能过度烷基化生成季铵盐)

  2. 酰基化反应：
    RNH2+R’COCl→RNHCOR′+HCl \text{RNH}_2 + \text{R'COCl} \rightarrow \text{RNHCOR}' + \text{HCl} RNH2​+R’COCl→RNHCOR′+HCl

  3. 与亚硝酸反应：

    • 伯胺：
      RNH2+HNO2→ROH+N2 \text{RNH}_2 + \text{HNO}_2 \rightarrow \text{ROH} + \text{N}_2 RNH2​+HNO2​→ROH+N2​
    • 仲胺：
      R2NH+HNO2→R2N-N=O \text{R}_2\text{NH} + \text{HNO}_2 \rightarrow \text{R}_2\text{N-N=O} R2​NH+HNO2​→R2​N-N=O
    • 叔胺：不反应

  4. 氧化反应：

    • 脂肪胺氧化复杂
    • 芳香胺易氧化，如苯胺氧化为苯醌

2. 醛与酮

2.1 结构与命名

2.1.1 结构特征

  • 羰基(C=O)为平面结构，sp2^22杂化
  • 键角约120°
  • 偶极矩：2.3-2.8D

2.1.2 命名规则

  1. 醛的命名：

    • 选择含醛基的最长碳链
    • 编号从醛基碳开始
    • 后缀"-al"
    • 例：CH3_33​CH2_22​CHO → 丙醛

  2. 酮的命名：

    • 选择含羰基的最长碳链
    • 编号使羰基位置最小
    • 后缀"-one"
    • 例：CH3_33​COCH2_22​CH3_33​ → 丁-2-酮

  3. 环酮命名：

    • 以"环"开头
    • 例：=O=O=O → 环己酮

  4. 芳香醛酮命名：

    • 苯甲醛(C6_66​H5_55​CHO)
    • 苯乙酮(C6_66​H5_55​COCH3_33​)

2.2 物理性质

2.2.1 沸点

  • 高于同碳数烷烃，低于醇
  • 例：
    • 丙烷(-42°C) < 丙酮(56°C) < 丙醇(97°C)

2.2.2 溶解性

  • 低级醛酮(C1_11​-C4_44​)与水混溶
  • 随碳数增加水溶性降低

2.2.3 光谱特征

  • IR：C=O伸缩振动~1700cm−1^{-1}−1
  • 1^11H NMR：醛基质子~9-10ppm

2.3 化学性质

2.3.1 亲核加成反应

  1. 与HCN加成：
    R2C=O+HCN→R2C(CN)OH \text{R}_2\text{C=O} + \text{HCN} \rightarrow \text{R}_2\text{C(CN)OH} R2​C=O+HCN→R2​C(CN)OH

  2. 与醇加成：

    • 半缩醛：
      RCHO+R’OH→H+RCH(OH)OR′ \text{RCHO} + \text{R'OH} \xrightarrow{\text{H}^+} \text{RCH(OH)OR}' RCHO+R’OHH+​RCH(OH)OR′
    • 缩醛：
      RCH(OH)OR′+R’OH→H+RCH(OR′)2 \text{RCH(OH)OR}' + \text{R'OH} \xrightarrow{\text{H}^+} \text{RCH(OR}')_2 RCH(OH)OR′+R’OHH+​RCH(OR′)2​

  3. 与胺衍生物反应：

    • 肼：
      R2C=O+H2NNH2→R2C=NNH2 \text{R}_2\text{C=O} + \text{H}_2\text{NNH}_2 \rightarrow \text{R}_2\text{C=NNH}_2 R2​C=O+H2​NNH2​→R2​C=NNH2​
    • 羟胺：
      R2C=O+H2NOH→R2C=NOH \text{R}_2\text{C=O} + \text{H}_2\text{NOH} \rightarrow \text{R}_2\text{C=NOH} R2​C=O+H2​NOH→R2​C=NOH

2.3.2 氧化还原反应

  1. 醛的氧化：

    • Tollens试剂：
      RCHO+2[Ag(NH3)2]++3OH−→RCOO−+2Ag+4NH3+2H2O \text{RCHO} + 2[\text{Ag(NH}_3\text{)}_2]^+ + 3\text{OH}^- \rightarrow \text{RCOO}^- + 2\text{Ag} + 4\text{NH}_3 + 2\text{H}_2\text{O} RCHO+2[Ag(NH3​)2​]++3OH−→RCOO−+2Ag+4NH3​+2H2​O
    • Fehling试剂：
      RCHO+2Cu2++5OH−→RCOO−+Cu2O+3H2O \text{RCHO} + 2\text{Cu}^{2+} + 5\text{OH}^- \rightarrow \text{RCOO}^- + \text{Cu}_2\text{O} + 3\text{H}_2\text{O} RCHO+2Cu2++5OH−→RCOO−+Cu2​O+3H2​O

  2. 还原反应：

    • 催化氢化：
      RCHO+H2→NiRCH2OH \text{RCHO} + \text{H}_2 \xrightarrow{\text{Ni}} \text{RCH}_2\text{OH} RCHO+H2​Ni​RCH2​OH
    • 金属氢化物：
      R2C=O+LiAlH4→R2CHOH \text{R}_2\text{C=O} + \text{LiAlH}_4 \rightarrow \text{R}_2\text{CHOH} R2​C=O+LiAlH4​→R2​CHOH

2.3.3 α-氢的反应

  1. 卤代反应：
    CH3COCH3+Br2→H+CH3COCH2Br \text{CH}_3\text{COCH}_3 + \text{Br}_2 \xrightarrow{\text{H}^+} \text{CH}_3\text{COCH}_2\text{Br} CH3​COCH3​+Br2​H+​CH3​COCH2​Br

  2. 羟醛缩合：
    2CH3CHO→OH−CH3CH(OH)CH2CHO 2\text{CH}_3\text{CHO} \xrightarrow{\text{OH}^-} \text{CH}_3\text{CH(OH)CH}_2\text{CHO} 2CH3​CHOOH−​CH3​CH(OH)CH2​CHO

3. 重要反应机理

3.1 Grignard反应

  1. 与甲醛反应：
    RMgX+HCHO→RCH2OMgX→H3O+RCH2OH \text{RMgX} + \text{HCHO} \rightarrow \text{RCH}_2\text{OMgX} \xrightarrow{\text{H}_3\text{O}^+} \text{RCH}_2\text{OH} RMgX+HCHO→RCH2​OMgXH3​O+​RCH2​OH

  2. 与其他醛反应：
    RMgX+R’CHO→RR’CHOMgX→H3O+RR’CHOH \text{RMgX} + \text{R'CHO} \rightarrow \text{RR'CHOMgX} \xrightarrow{\text{H}_3\text{O}^+} \text{RR'CHOH} RMgX+R’CHO→RR’CHOMgXH3​O+​RR’CHOH

  3. 与酮反应：
    RMgX+R2′CO→RR2′COMgX→H3O+RR2′COH \text{RMgX} + \text{R}'_2\text{CO} \rightarrow \text{RR}'_2\text{COMgX} \xrightarrow{\text{H}_3\text{O}^+} \text{RR}'_2\text{COH} RMgX+R2′​CO→RR2′​COMgXH3​O+​RR2′​COH

3.2 缩醛形成机理

  1. 质子化：
    RCHO+H+→RCH=OH+ \text{RCHO} + \text{H}^+ \rightarrow \text{RCH=OH}^+ RCHO+H+→RCH=OH+

  2. 亲核加成：
    RCH=OH++R’OH→RCH(OH)OR′ \text{RCH=OH}^+ + \text{R'OH} \rightarrow \text{RCH(OH)OR}' RCH=OH++R’OH→RCH(OH)OR′

  3. 二次质子化与取代：
    RCH(OH)OR′+H+→RCH(OR′)2 \text{RCH(OH)OR}' + \text{H}^+ \rightarrow \text{RCH(OR}')_2 RCH(OH)OR′+H+→RCH(OR′)2​

3.3 亚胺形成机理

  1. 亲核加成：
    R2C=O+H2NR′→R2C(OH)NHR′ \text{R}_2\text{C=O} + \text{H}_2\text{NR}' \rightarrow \text{R}_2\text{C(OH)NHR}' R2​C=O+H2​NR′→R2​C(OH)NHR′

  2. 脱水：
    R2C(OH)NHR′→H+R2C=NR′+H2O \text{R}_2\text{C(OH)NHR}' \xrightarrow{\text{H}^+} \text{R}_2\text{C=NR}' + \text{H}_2\text{O} R2​C(OH)NHR′H+​R2​C=NR′+H2​O

4. 应用与合成实例

4.1 布洛芬合成路线

  1. Friedel-Crafts酰化：
    C6H6+(CH3CO)2O→AlCl3C6H5COCH3 \text{C}_6\text{H}_6 + (\text{CH}_3\text{CO})_2\text{O} \xrightarrow{\text{AlCl}_3} \text{C}_6\text{H}_5\text{COCH}_3 C6​H6​+(CH3​CO)2​OAlCl3​​C6​H5​COCH3​

  2. 侧链延伸：
    C6H5COCH3+ClCH2CO2Et→C6H5COCH2CH2CO2Et \text{C}_6\text{H}_5\text{COCH}_3 + \text{ClCH}_2\text{CO}_2\text{Et} \rightarrow \text{C}_6\text{H}_5\text{COCH}_2\text{CH}_2\text{CO}_2\text{Et} C6​H5​COCH3​+ClCH2​CO2​Et→C6​H5​COCH2​CH2​CO2​Et

  3. 脱羧与还原：
    C6H5COCH2CH3→NaBH4C6H5CH(OH)CH2CH3 \text{C}_6\text{H}_5\text{COCH}_2\text{CH}_3 \xrightarrow{\text{NaBH}_4} \text{C}_6\text{H}_5\text{CH(OH)CH}_2\text{CH}_3 C6​H5​COCH2​CH3​NaBH4​​C6​H5​CH(OH)CH2​CH3​

4.2 阿司匹林合成

  1. Kolbe-Schmitt反应：
    C6H5OH+CO2→NaOHHO-C6H4-COOH \text{C}_6\text{H}_5\text{OH} + \text{CO}_2 \xrightarrow{\text{NaOH}} \text{HO-C}_6\text{H}_4\text{-COOH} C6​H5​OH+CO2​NaOH​HO-C6​H4​-COOH

  2. 乙酰化：
    HO-C6H4-COOH+(CH3CO)2O→CH3COO-C6H4-COOH \text{HO-C}_6\text{H}_4\text{-COOH} + (\text{CH}_3\text{CO})_2\text{O} \rightarrow \text{CH}_3\text{COO-C}_6\text{H}_4\text{-COOH} HO-C6​H4​-COOH+(CH3​CO)2​O→CH3​COO-C6​H4​-COOH

4.3 其他重要药物

  1. 氯苯那敏(抗组胺药)：
    • 结构：含吡啶环的叔胺
    • 合成关键步骤：
      吡啶→CH3ClN-甲基吡啶盐→NaCN2-氰基吡啶 \text{吡啶} \xrightarrow{\text{CH}_3\text{Cl}} \text{N-甲基吡啶盐} \xrightarrow{\text{NaCN}} \text{2-氰基吡啶} 吡啶CH3​Cl​N-甲基吡啶盐NaCN​2-氰基吡啶
