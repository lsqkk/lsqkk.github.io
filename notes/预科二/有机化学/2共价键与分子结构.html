共价键形成和参数

1. 共价键模型

1.1 共价键的形成

  • 定义：非金属原子间通过共享电子形成共价键，电子密度集中在两核之间。
  • 能量最低：分子通过共价键使体系能量达到最低。
  • 键的表示：
    • 单键（Single bond）：—（键级=1）
    • 双键（Double bond）：=（键级=2）
    • 三键（Triple bond）：≡（键级=3）

1.2 键级、键能、键长

  • 键级（Bond Order）：共享电子对数，键级↑ → 键能↑，键长↓。
  • 键能（Bond Energy, BE）：
    • 定义：1 mol气态分子断裂化学键所需的能量（ΔH⁰ > 0）。
    • 公式：
      ΔrHm⊖=∑E断键−∑E成键 \Delta_r H_m^\ominus = \sum E_{\text{断键}} - \sum E_{\text{成键}} Δr​Hm⊖​=∑E断键​−∑E成键​
    • 单位：kJ/mol。
  • 键长（Bond Length）：两原子核间距离，键级↑ → 键长↓。

2. 电负性与键极性

2.1 电负性（Electronegativity, EN）

  • 定义：原子吸引共享电子的能力（Pauling标度，F=4.0）。
  • 规律：
    • 同一周期：从左到右↑
    • 同一主族：从下到上↑
  • 键类型判断：
    • ΔEN ≥ 1.7 → 离子键（Ionic Bond）
    • ΔEN < 1.7 → 共价键（Covalent Bond）
      • 极性共价键（Polar Covalent）：ΔEN = 0.7–1.6
      • 非极性共价键（Nonpolar Covalent）：ΔEN = 0

示例

键型 ΔEN 键性质
H-H 0 非极性共价键
H-F 1.9 极性共价键
LiCl 2.0 离子键（92%离子性）

2.2 键的可极化性（Polarizability）

  • 定义：外电场下键极性变化的程度。
    键的可极化性用极化度来度量，它表示成键电子被成键原子核约束的相对程度。极化度与成键原子的体积、电负性、键的种类以及外电场的强度有关。
  • 影响因素：原子半径↑ → 极化性↑；电负性↑ → 极化性↓。
  • 示例：C-I > C-Br > C-Cl（极化性递增，极性递减）。

3. 分子极性

3.1 分子极性的判断

  • 偶极矩（Dipole Moment, μ）：
    μ=q×d(单位：D，1D=3.34×10−30C⋅m) \mu = q \times d \quad (单位：D，1 D = 3.34×10⁻³⁰ C·m) μ=q×d(单位：D，1D=3.34×10−30C⋅m)
  • 分子形状的影响：
    • 非极性分子：电荷中心重合（如CO₂，线性结构，μ=0）。
    • 极性分子：电荷中心不重合（如H₂O，V型，μ=1.85 D）。

H₂O 极性键 V型 极性
CCl₄ 极性键 四面体 非极性
CHCl₃ 极性键 四面体不对称 极性

4. VSEPR理论与分子形状

4.1 理论核心

  • 价层电子对互斥（Valence-Shell Electron-Pair Repulsion, VSEPR）：电子对（成键+孤对）互相排斥，形成最大夹角。
  • 步骤：

    1. 确定中心原子价层电子对数（VPN）：
      VPN=12(中心原子价电子数+净接收电子数) \text{VPN} = \frac{1}{2} (\text{中心原子价电子数} + \text{净接收电子数}) VPN=21​(中心原子价电子数+净接收电子数)

    2. 根据VPN确定电子对排列和分子形状。

      1. 从分子式中写出刘易斯结构，以显示原子的相对位置和电子基团的数量。
      2. 通过计算中心原子周围的所有电子基（键和非键）来分配电子基团排列。
      3. 根据电子基排列预测理想键角以及孤对或双键引起的任何偏差的影响。
      4. 通过分别计算键基团和非键基团来绘制和命名分子形状。

    3. 不同电子对之间的排斥能力顺序是：

孤对电子 - 孤对电子 > 孤对电子 - 成键电子 > 成键电子 - 成键电子 。

5. 物理性质与化学反应

  • 共价物质性质：
    • 分子晶体（如H₂O）：低熔点，弱分子间作用力。
    • 网状共价晶体（如金刚石）：高熔点，强共价键网络。
  • 化学反应本质：旧键断裂与新键形成，能量变化由键能决定。

重点总结

  1. 共价键参数：键级、键能、键长的关系。
  2. 电负性与键极性：ΔEN决定键类型及极性大小。
  3. 分子极性：由分子形状和键极性共同决定。
  4. VSEPR理论：通过价层电子对数预测分子几何构型。
  5. 杂化轨道：sp³、sp²、sp杂化对应不同分子形状。
